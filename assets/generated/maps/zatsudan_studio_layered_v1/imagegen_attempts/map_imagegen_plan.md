# Zatsudan Studio Image-Generation Plan

Pipeline:

- `map_mode`: `scene_mode`
- `visual_model`: `layered_raster`
- `runtime_object_model`: `separate_props`
- `collision_model`: `precise_shapes`
- `engine_target`: Godot project-native

Asset contract:

1. Generate a foundation-only floor image at `2200x1500`.
2. Reject it if it contains text, letters, UI, characters, props, furniture, or infographic/poster composition.
3. If accepted, copy it over `zatsudan_studio_floor_2200x1500.png`.
4. Keep props in `zatsudan_studio_props_2200x1500.png` or regenerate them as separate prop assets/layers.
5. Keep collision in `manifest.json` and sync to `scripts/systems/map_background_system.gd`.
6. Run `tools/check_map_background_manifest.py` before integration is considered done.

Current fallback:

- Use the procedural layered map generated by `tools/generate_zatsudan_studio_layered_bg.py` until generated art passes the contract.
